# CSS2 UQ Project

This is the official repo for CSS2 UQ project. 

The data generation is in the data_work directory, this is already been run, though we may do so again in the future if the dataset is not of high quality.

Every model experiment should be placed in the UQ directory, where each person can create their own subdirectory, either their name or the name of their experiment (architecture change, algorithm change etc). 

Contribution guide:
- Please clone the repo and checkout your own branch (if you are unfamiliar with git, I suggest looking up a guide or asking a coding agent to assist you).
- All changes that you make must be on your own branch first to prevent accidental conflicts with other team members work
- If you are satisfied with your branch and the work you did on it, you may submit a pull request, which team members can review and comment on before merging.
